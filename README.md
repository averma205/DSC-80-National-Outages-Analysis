<p><b>Stubb Place</b> is a small settlement in western <a href="/wiki/Cumbria" title="Cumbria">Cumbria</a>, England. Known by locals in Waberthwaite as marshside, due to the name of the nearby cottages. It lies within the <a href="/wiki/Lake_District_National_Park" class="mw-redirect" title="Lake District National Park">Lake District National Park</a>.
</p>
